# oxmpl-js

JavaScript and WebAssembly (WASM) bindings for the [OxMPL](https://github.com/juniorsundar/oxmpl) motion planning library.

## Documentation

For installation instructions, API reference, and usage examples, please refer to the **[JavaScript API section](https://juniorsundar.github.io/oxmpl/js_api.html)** in the OxMPL Book.
